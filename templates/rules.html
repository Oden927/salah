Règles du jeu Loup-Garou

Objectif du jeu

Le but du jeu dépend de votre rôle :

  • Villageois : Éliminer tous les loups-garous.
  • Loups-Garous : Éliminer tous les villageois.
  • Personnages spéciaux : Suivre leurs objectifs uniques pour aider leur camp.

Déroulement d'une partie

1. Phase de nuit

Les loups-garous choisissent une victime. Les personnages spéciaux (Voyante, Sorcière, etc.) peuvent utiliser leurs pouvoirs.

2. Phase de jour

Les joueurs discutent pour identifier les loups-garous et votent pour éliminer un suspect.

Rôles disponibles

  • Villageois : Aucun pouvoir spécial, doit voter stratégiquement.
  • Loups-Garous : Se réunissent la nuit pour éliminer un joueur.
  • Voyante : Peut connaître le rôle d'un joueur chaque nuit.
  • Sorcière : Dispose d'une potion de vie et d'une potion de mort.
  • Cupidon : Lie deux joueurs qui deviennent amoureux.
  • Chasseur : Peut éliminer un joueur en mourant.

Conditions de victoire

  • Villageois : Tous les loups-garous sont éliminés.
  • Loups-Garous : Le nombre de loups-garous est égal ou supérieur à celui des villageois.
  • Amoureux : Si tous les autres joueurs sont éliminés, les amoureux gagnent.
